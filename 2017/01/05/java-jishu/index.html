一、Java的基础知识

1、集合的概念

集合是具有同种属性的对象的集合。是一个工具类。

2、与数组相比

数组：数组长度固定，且只能通过下标访问。
集合：长度任意伸缩，且有的集合可以通过任意属性的对象去找所映射的对象，
集合根接口：Collection包括三个子接口（list、queue、set） Map包括HashMap.

(序列）List的实现类：ArrayList 是一个数组序列（有序可重复）
（队列）Queue的实现类：LinkedList
（集）Set实现类：HashSet
增：add方法，addAll方法。 删：remove（）。改：set（）。查：get（）
三种循环：for循环、iterator迭代器循环、foreach循环
